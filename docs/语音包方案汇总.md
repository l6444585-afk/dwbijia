# 语音包（TTS）方案汇总

> 适用场景：冥想/助眠App，需要至少10种不同声音（5男5女），要求音色柔和自然

---

## 一、免费/开源方案

### 1. Edge TTS（推荐快速起步）

- **特点**：微软免费TTS，20+种中文声音（男女都有），质量接近商业级
- **安装**：`pip install edge-tts`
- **适合冥想的声音**：
  - 女声：`zh-CN-XiaoxiaoNeural`（温柔）、`zh-CN-XiaomoNeural`（知性）、`zh-CN-XiaoshuangNeural`（甜美）、`zh-CN-XiaoruiNeural`（沉稳）、`zh-CN-XiaoyiNeural`（活力）
  - 男声：`zh-CN-YunxiNeural`（阳光）、`zh-CN-YunjianNeural`（磁性）、`zh-CN-YunxiaNeural`（少年）、`zh-CN-YunyeNeural`（故事）、`zh-CN-YunzeNeural`（沉稳）
- **用法示例**：
  ```bash
  edge-tts --voice zh-CN-XiaoxiaoNeural --text "闭上眼睛，深呼吸" --write-media output.mp3
  ```
- **地址**：https://github.com/rany2/edge-tts

### 2. ChatTTS（最自然的中文开源TTS）

- **特点**：专为对话场景设计，支持笑声、停顿等情感控制，中文极其自然
- **适合场景**：AI冥想引导、助眠故事朗读
- **安装**：
  ```bash
  pip install ChatTTS
  ```
- **亮点**：可通过随机种子生成不同声音，理论上可以生成无限种音色
- **地址**：https://github.com/2noise/ChatTTS

### 3. GPT-SoVITS（声音克隆）

- **特点**：只需5秒音频样本即可克隆任何人的声音
- **适合场景**：打造独家专属音色，竞品无法复制
- **安装**：需要GPU，建议用Colab或本地3060以上显卡
- **地址**：https://github.com/RVC-Boss/GPT-SoVITS

### 4. Fish Audio

- **特点**：开源TTS+声音克隆，在线体验无需部署
- **免费额度**：注册送一定额度
- **地址**：https://fish.audio

## 二、商业方案

| 项目 | 说明 |
|------|------|
| 中文声音数 | 100+种（含情感音色） |

## 三、推荐方案组合

| 阶段 | 方案 | 理由 |
|------|------|------|
| **MVP快速验证** | Edge TTS | 免费、即开即用、10种声音直接可用 |
| **正式上线** | 讯飞开放平台 | 中文最强、音色最多、有冥想专用音色 |
| **差异化竞争** | GPT-SoVITS 声音克隆 | 打造独家音色，竞品无法模仿 |
