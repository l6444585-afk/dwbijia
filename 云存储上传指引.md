# 易眠AI - 白噪音云存储上传指引

## 任务说明
将白噪音和背景音乐的 MP3 文件上传到微信云存储，让小程序可以在线播放。

## 操作步骤

### 1. 打开云开发控制台
- 打开**微信开发者工具**
- 点击左上角**云开发**按钮
- 进入**存储**标签页

### 2. 创建目录结构
在云存储中创建以下两个文件夹：
```
audio/
  whitenoise/    ← 白噪音（37个文件）
  bgmusic/       ← 背景音乐（12个文件）
```

## 常见问题

**Q: 上传很慢怎么办？**
A: 总共约 180MB，建议在网速好的环境上传，一次上传几个文件。

**Q: 文件名有问题怎么办？**
A: 确保文件名和上面列表完全一致。如果素材文件夹里文件名不同，需要改名后再上传。

**Q: 上传失败？**
A: 检查云开发环境的存储额度，免费额度 5GB，足够用。
